### Q:在Terminal中输入Python，调用的是系统自带的Py2.7
	A:Anaconda Initiate将环境变量复位了
	配置anaconda环境：
	vim ~/.bashrc
	最后一行添加：export PATH=/home/XXX/anaconda3/bin:$PATH
	XXX是你的用户名
	保存并： source ~/.bashrc
